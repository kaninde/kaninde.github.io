# Acerca de Pix QR Code Generator

Bienvenido a **Pix QR Code Generator**, la aplicación definitiva para generar y gestionar códigos QR PIX, diseñada para simplificar tus pagos. Con una interfaz simple y funcional, Pix QR Code Generator te ayuda a crear códigos QR PIX a partir de claves o códigos Copiar y Pegar, organizar tus contactos y llevar el control de tu historial de transacciones.

---

## Funcionalidades

### Gratuitas

- **Generación de QR Code PIX:** Crea códigos QR con un solo toque desde el portapapeles o escribiendo manualmente. Validación completa de códigos EMV (Copiar y Pegar).
- **Historial de PIX:** Visualiza los códigos QR PIX generados anteriormente, con límites de elementos en el plan gratuito.
- **Contactos y Claves PIX:** Organiza contactos y asocia claves PIX para generar códigos más fácilmente.
- **Mis Cuentas:** Registra cuentas bancarias e instituciones para acceso rápido a tus claves PIX.
- **Estadísticas básicas:** Consulta resúmenes de tu uso de PIX.
- **Soporte Offline:** Funciona completamente sin conexión a internet.
- **Diseño ligero y rápido:** Interfaz minimalista enfocada en simplicidad y usabilidad.

### Premium (Suscripción)

- **Estadísticas avanzadas:** Análisis detallados, insights inteligentes y patrones temporales.
- **Exportar/Importar Datos:** Realiza copias de seguridad y migra tus datos entre dispositivos.
- **Plantillas de QR Code:** Modelos listos para acelerar la creación de códigos PIX.
- **Campos avanzados en la creación:** Fecha de expiración configurable, GPS para ciudad y otras funciones avanzadas.
- **Historial ilimitado:** Guarda y marca como favorito todos los elementos que necesites.
- **Temas personalizados:** Personaliza la apariencia de la aplicación.

### Anuncios (AdSense)

Algunas funcionalidades pueden utilizarse de forma **temporal** al ver un anuncio. Por ejemplo: guardar un código QR en el historial, marcar un elemento como favorito o exportar un contacto. Al ver el anuncio, la acción se desbloquea en ese momento sin necesidad de suscripción premium. El uso de anuncios es opcional y no afecta las funcionalidades totalmente gratuitas.

---

## Nuestra Misión

Nuestro objetivo es facilitar la generación de códigos QR PIX y ayudarte a mantenerte organizado. Pix QR Code Generator es más que un simple generador: es una herramienta para centralizar tus claves PIX, contactos e historial en un solo lugar, haciendo que cada pago sea más ágil y práctico.

---

## Preguntas Frecuentes (FAQ)

**¿Pix QR Code Generator necesita internet para funcionar?**  
No. La aplicación funciona completamente offline. La conexión a internet se utiliza únicamente para funciones opcionales, como anuncios o servicios premium.

**¿Necesito crear una cuenta para usar Pix QR Code Generator?**  
No. La aplicación no requiere creación de cuenta. Tus datos se almacenan localmente en tu dispositivo.

**¿Perderé mis datos si desinstalo la aplicación?**  
Sí. Como los datos se almacenan localmente, al desinstalar la aplicación se eliminarán permanentemente tu historial, contactos y configuraciones.

**¿Pix QR Code Generator es adecuado para principiantes?**  
Sí. La aplicación está diseñada para todo tipo de usuarios, desde quienes usan PIX por primera vez hasta quienes realizan pagos con frecuencia.

**¿Pix QR Code Generator procesa pagos?**  
No. La aplicación solo genera códigos QR PIX. El pago se realiza en la aplicación de tu banco, que escanea el código QR o el código Copiar y Pegar generado.

**¿Qué incluye el plan premium?**  
El plan premium ofrece funciones avanzadas como estadísticas detalladas, insights inteligentes, plantillas de QR Code y exportación/importación de datos para copias de seguridad o migración entre dispositivos.
